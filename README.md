# Graph Traversal Project (DFS & BFS)

This project demonstrates Depth First Search (DFS) and Breadth First Search (BFS)
traversal techniques using an undirected graph.

## Features
- Add edges to graph
- BFS traversal
- DFS traversal
- Simple Java implementation
